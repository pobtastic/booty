CD6F: Set Default Keyboard Inputs
Used by the routines at DisplayIntroductionScreen, SetCursorJoystick and SetDefaultKeys.
See Table_UserDefinedKeyMap for the corresponding keymap table.
SetDefaultKeyboardInputs CD6F LD HL,$5BEB Write "5" (04) to *UserDefinedKeys_Left.
CD72 LD (HL),$04
CD74 INC HL Write "8" (13) to *UserDefinedKeys_Right.
CD75 LD (HL),$13
CD77 INC HL Write "7" (0B) to *UserDefinedKeys_Up.
CD78 LD (HL),$0B
CD7A INC HL Write "6" (03) to *UserDefinedKeys_Down.
CD7B LD (HL),$03
CD7D INC HL Write "A" (26) to *UserDefinedKeys_Fire.
CD7E LD (HL),$26
CD80 LD A,$24 Write Keyboard Input (24) to *ControlMethod.
CD82 LD ($5BEA),A
CD85 RET Return.
