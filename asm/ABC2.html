ABC2: Move Room Data
Used by the routines at UnpackAllRooms, UnpackRoom and PopulateCurrentRoomBuffersAndReferences.
Input
B Loop counter
DE The room buffer target destination
HL Pointer to the room data we want to copy
MoveRoomData ABC2 PUSH BC Stash the loop counter on the stack.
ABC3 LD A,(HL) Fetch a byte from the source room data pointer and store it in A.
Have we finished with everything?
ABC4 CP $FF If the terminator character (FF) has been reached jump to MoveRoomData_Next.
ABC6 JR Z,MoveRoomData_Next
Handle copying the data from the source room data to the target room buffer.
MoveRoomData_Loop ABC8 LD (DE),A Write the room data byte to the room buffer target destination.
ABC9 INC HL Increment the source room data pointer by one.
ABCA INC DE Increment the room buffer target destination by one.
ABCB LD A,(HL) Fetch a byte from the source room data pointer and store it in A.
ABCC DJNZ MoveRoomData_Loop Decrease counter by one and loop back to MoveRoomData_Loop until counter is zero.
Using the same counter as on entry to the routine, start the process again.
ABCE POP BC Restore the original loop counter from the stack.
ABCF JR MoveRoomData Jump to MoveRoomData.
This cycle is now over, so store the terminator in the room buffer, increment both pointers ready for the next call to this routine and finally, tidy up the stack.
MoveRoomData_Next ABD1 LD (DE),A Write the termination character to the room buffer target destination.
ABD2 INC DE Increment the room buffer target destination by one.
ABD3 INC HL Increment the source room data pointer by one.
ABD4 POP BC Housekeeping; discard the loop counter from the stack.
ABD5 RET Return.
